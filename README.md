# bus-leyu-guide

本仓库用于归档和发布多个独立 HTML 页面。这些页面以静态 HTML 文件的形式存在，可直接通过浏览器打开查看，无需后端服务支持。

## 项目简介

bus-leyu-guide 是一个面向通用目的的 HTML 页面集合。每个页面独立存在，内容互不依赖。仓库本身不针对特定网站或服务，仅作为页面存档与展示的载体。

## 目录说明

```
bus-leyu-guide/
├── index.html          # 仓库入口页面（可选）
├── pages/              # 存放所有独立 HTML 页面
│   ├── example1.html
│   ├── example2.html
│   └── ...
├── assets/             # 页面引用的静态资源（可选）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：主要页面目录，每个 HTML 文件为一个独立页面。
- `assets/`：存放页面所需的样式、脚本、图片等资源文件（如有需要）。

## 页面归档说明

- 所有页面均为静态 HTML，兼容主流现代浏览器。
- 页面之间无跨域或后端依赖，可直接双击打开或部署至任意静态服务器。
- 归档内容可能随时间更新或调整，不保证长期不变。

## 维护说明

- 欢迎通过提交 Issue 或 Pull Request 提出建议或修正。
- 新增页面请放入 `pages/` 目录，并保持文件命名简洁清晰。
- 如需引用外部资源，请尽量使用相对路径或兼容性良好的 CDN 链接。
